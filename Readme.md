# 🔐 PyPassword Generator

A simple Python-based password generator that creates strong and random passwords using letters, symbols, and numbers.

---

## 🚀 Features
- Generate random passwords
- User can choose:
  - Number of letters
  - Number of symbols
  - Number of numbers
- Fully customizable password
- Randomized order for better security

---

## 🛠️ Technologies Used
- Python
- random module

---

## 📌 How It Works
1. User inputs:
   - Number of letters
   - Number of symbols
   - Number of numbers
2. Program randomly selects characters
3. Shuffles them to create a secure password
4. Displays final password

---

## ▶️ How to Run
1. Clone the repository
2. Run the Python file:
